# Contact Organizer App

A Flutter-based Contact Organizer App that combines local storage with SQLite and cloud synchronization via Firebase. Manage, categorize, and access your contacts seamlessly with offline functionality and advanced search options.

---

## ðŸš€ Features

### 1. Efficient Contact Management
- **Fields**: Name, Phone Number, Email, Category (e.g., Family, Friends, Work), and Favorite (Yes/No).
- **CRUD Operations**:
  - Add new contacts with category selection.
  - View contacts categorized by groups (e.g., Family, Friends, Work).
  - Edit contact details, including category and favorite status.
  - Delete contacts when needed.

### 2. Cloud Backup and Synchronization
- **Firebase Authentication**: Secure login using email and password.
- **Firestore Integration**:
  - Upload contact data to Firestore for cloud backup.
  - Sync and download contacts to your local SQLite database.

### 3. Advanced Search and Filters
- Search contacts by name.
- Filter contacts by categories (e.g., Family, Friends, Work).
- Sort contacts by Favorites to quickly find important ones.

### 4. Offline Access with Auto-Sync
- Full offline functionality using SQLite for local storage.
- Automatically sync data with Firebase when the device reconnects to the internet.

### 5. Intuitive User Interface
- **Categorized ListView**: Displays contacts in a clear and organized manner.
- **Floating Action Button (FAB)**: Quickly add new contacts.
- **Visual Indicators**:
  - Icons or colors to differentiate contact categories.
  - Star or highlight favorite contacts for quick identification.
